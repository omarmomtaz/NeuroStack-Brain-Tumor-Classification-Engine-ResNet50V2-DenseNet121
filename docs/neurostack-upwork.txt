OVERVIEW
────────────────────────────────────────────────────────────────

NeuroStack is a production-grade deep learning system for brain tumor classification from MRI scans. Built to distinguish Glioma (malignant, aggressive) from Meningioma (typically benign) with clinical-grade precision, the system achieved 99.88% accuracy, 100% specificity, and zero false positives on a held-out test set of 800 scans.

This project demonstrates end-to-end ownership of a high-stakes medical AI pipeline — from raw data through architecture design, training, evaluation, and visualization.

SYSTEM ARCHITECTURE
────────────────────────────────────────────────────────────────

Two-level stacked ensemble:

LEVEL 1 — Base Models (10 total)
• ResNet50V2Classifier — 5-fold cross-validation
• DenseNet121Classifier — 5-fold cross-validation
• Each trained independently on 2,800 MRI scans
• Each produces calibrated class probabilities

LEVEL 2 — Meta-Learner
• Input: (N, 20) stacked probability matrix
• StandardScaler normalization
• Logistic Regression trained on out-of-fold predictions
• Output: final binary prediction + confidence score

Key design decisions:
• Stacking over majority voting — preserves full probability signal
• Heterogeneous architectures — ResNet's global texture + DenseNet's feature reuse provide complementary failure modes
• MixUp training — smoother decision boundaries, reduced overconfidence
• Stratified 5-fold CV — zero data leakage, rigorous generalization estimate
